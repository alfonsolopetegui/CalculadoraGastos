Calculadora de Gastos

Una aplicación simple para realizar un seguimiento de tus gastos diarios y generar visualizaciones útiles.

Funcionalidades

Registro de gastos: Agregar nuevos gastos con nombre, cantidad, categoría y fecha.
Visualización de gastos: Ver una tabla de gastos ordenada por fecha y filtrada por categoría y rango de fechas.
Gráfico de pastel: Visualizar un gráfico de pastel que muestra la distribución de gastos por categoría.
Paginación: Navegar a través de los gastos paginados.

Instalación
Clona este repositorio en tu máquina local.

git clone https://github.com/alfonsolopetegui/CalculadoraGastos.git

instala TypeScript: npm install -g typescript

compila a JS: tsc app.ts -w

Abre el archivo index.html en tu navegador web.

Para agregar un nuevo gasto, completa el formulario en la parte superior de la página y haz clic en "Agregar".
Utiliza los filtros de fecha y categoría para refinar la visualización de los gastos.
Haz clic en los botones de paginación para navegar a través de los diferentes conjuntos de gastos.
